# Friends_Dashboard_Tableau
This Tableau dashboard provides an interactive and comprehensive analysis of the Friends TV series, focusing on episode-level metrics across all ten seasons. It is designed to help users explore patterns in episode ratings, votes, directors, and season-wise performance through visually engaging charts.

This interactive Tableau dashboard provides a comprehensive analysis of all ten seasons of the Friends TV series. It highlights key episode-level insights through visually engaging and interactive components. The dashboard showcases overall trends in audience ratings, vote counts, episode durations, and directing patterns, enabling users to easily explore the performance of the series across its entire run.

Key features include:

Episode Insight Panel: Displays detailed information such as season, episode number, release year, duration, total stars, and viewer votes.

Directors Word Cloud: Highlights the most frequently featured directors and their contribution to the show.

Season-wise Episode Distribution: Visualizes the number of episodes across all seasons for quick comparison.

Votes & Ratings Analysis: Provides season and episode-level vote counts and a box-plot distribution of star ratings to identify trends, outliers, and audience engagement patterns.

Episode Description & Visual Context: Includes a synopsis and an image to provide narrative context for each selected episode.

This dashboard demonstrates strong storytelling with data, effective use of visual analytics, and the ability to convert raw information into meaningful insights.
